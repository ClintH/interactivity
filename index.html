interactivity

A collection of interactivity-oriented samples for student use. Most of these are drawn from existing examples, but simplified.

Many of these samples will require additional steps to work. Each folder has a README.md with installation steps and links for further reading.

  1. Basics: starter skeleton for mobile and a watch UI
  2. Canvas: drawing visuals
  3. Camera: Computer vision, working with the camera
  4. Connectivity: Connecting to a microcontroller via serial and websockets
  5. Data: Persisting data within the browser or online
  6. Pointer events: touch, pressure
  7. Typography: basic techniques with type in particular variable typefaces
  8. Analog joystick: working with an Arduino component with web technologies
