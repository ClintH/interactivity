interactivity

A collection of interactivity-oriented samples for student use. Most of these are drawn from existing examples, but simplified.

Many of these samples will require additional steps to work. Each folder has a README.md with installation steps and links for further reading.

  1. Basics: starter skeleton for mobile and a watch UI
  2. Pointer events: touch, pressure
  3. Canvas: drawing visuals
  4. Typography: basic techniques with type in particular variable typefaces
  5. Data: sending data between sketches, and persisting online
  6. Camera: Computer vision, working with the camera
  7. Analog joystick: working with an Arduino component with web technologies
  8. Websockets: data to and from a microcontroller, Facebook Messenger
